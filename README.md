### AI Game Master Autonomous
This mod connects your Arma Reforger server to an external Python Backend powered by an AI (Llama 3.1). The AI acts as an autonomous Game Master (Zeus), evaluating the battlefield status and dynamically spawning units and vehicles to maintain tension, completely hands-free!

### What does the Smart Installer do?
To make this extremely easy, we have created an automatic Smart Installer in GitHub. By running the script, it will automatically:
1. Check for missing Python libraries and install them.
2. Detect if you have Ollama (the AI engine). If not, it installs it via PowerShell.
3. Automatically download the Llama 3.1 AI model (~4.7GB).
4. Generate the Backend server files, the Web Control Panel, and a one-click launcher.
